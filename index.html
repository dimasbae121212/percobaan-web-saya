Percobaan Membuat Web Hari Pertama

Apa Tujuan Saya Percobaan Ini?

Ini adalah langkah awal dalam mempelajari pengembangan web, kini dengan penambahan **CSS** untuk membuatnya lebih menarik secara visual. Tujuannya adalah untuk memahami bagaimana **HTML (Struktur)** dan **CSS (Gaya)** bekerja sama.

CSS memungkinkan kita mengontrol warna, *font*, tata letak, dan banyak aspek desain lainnya. Ini membuat laman web tidak hanya fungsional tetapi juga enak dilihat.

Contoh Elemen CSS yang Digunakan

  • Mengganti latar belakang laman menjadi **biru muda** (`background-color`).
  • Menambahkan **bayangan** (`box-shadow`) pada konten utama.
  • Mengganti jenis **font** menjadi Arial (`font-family`).
  • Memberi **warna biru** pada judul (`color`).
