Apa Tujuan Percobaan Ini?

Ini adalah langkah awal dalam mempelajari pengembangan web. Tujuannya adalah untuk memahami struktur dasar dari sebuah dokumen **HTML** (HyperText Markup Language).

Setiap laman web dibangun dengan elemen-elemen HTML seperti judul (<h1>), paragraf (<p>), dan elemen struktural lainnya.

Struktur Dasar

  • <!DOCTYPE html>: Mendefinisikan dokumen sebagai HTML5.
  • <html>: Elemen wadah utama untuk seluruh laman.
  • <head>: Berisi meta-informasi (judul laman, set karakter, dll.).
  • <body>: Berisi semua konten yang terlihat oleh pengguna.
